Nastavení

Propoj panel s GitHubem (jednou).

/

Jak získat token (30 vteřin)

1. Jdi na github.com/settings/tokens/new

2. Note: „DOLOŽENO redakce"

3. Expiration: „No expiration" (nebo 90 dní)

4. Zaškrtni repo (celý blok)

5. Generate token → zkopíruj → vlož sem
